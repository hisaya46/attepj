# Atte（アット）

Atteはある企業の勤怠管理システムです。登録したユーザー(従業員)の出退勤及び休憩の打刻ができます。

# 制作の背景と目的

模擬案件という形で開発しました。オンラインスクールの課題となっております。

人事評価及び従業員の勤怠管理を目的としています。

# 開発環境

* HTML/CSS
* Javascript
* PHP 7.4.21
* Laravel 8.83.6
* Laravel Breeze
* MAMP
* MySql 5.7.34

当アプリはOSはMac、エディターがVisual Studio Codeで開発しました。
